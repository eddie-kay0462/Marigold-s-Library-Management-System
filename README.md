# Marigold's Library Management System

A comprehensive library management system built with HTML, CSS, JavaScript, and PHP.

## Project Structure

```
├── assets/
│   ├── css/
│   ├── js/
│   └── images/
├── config/
│   └── database.php
├── includes/
│   ├── header.php
│   ├── footer.php
│   └── functions.php
├── admin/
│   ├── dashboard.php
│   ├── manage-books.php
│   ├── manage-students.php
│   └── manage-borrows.php
├── index.php
└── README.md
```

## Features

- Book Management
- Student Management
- Borrowing System
- Admin Dashboard
- Search Functionality
- User Authentication

## Setup Instructions

1. Clone the repository
2. Import the database schema from `Database Schema` file
3. Configure database connection in `config/database.php`
4. Set up a local web server (e.g., XAMPP, WAMP)
5. Access the application through your web browser

## Requirements

- PHP 7.4 or higher
- MySQL 5.7 or higher
- Web server (Apache/Nginx)
- Modern web browser
